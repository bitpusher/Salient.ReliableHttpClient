ItemName 
The name of the item involved in the current update, or null if a SimpleTableInfo was used to describe the related table.  
 
ItemPos 
Positional information for the item involved in the current update, i.e. the 1-based position of the item within the Group of items in the related table. 
 
NumFields 
The total number of fields contained in the subscription schema. In case a SimpleTableInfo was used to describe the related table, this number might not be known at subscription time. 
 
Snapshot 
Indicates whether or not the current update pertains to the item snapshot.
